Descripción detallada

Librería MicroPython para enviar lecturas a Firebase Realtime Database.

Esta clase permite autenticarse con Firebase mediante correo y contraseña, y enviar mediciones o datos arbitrarios a la ruta raw/ de la base de datos. Soporta conexión a Wi-Fi para dispositivos ESP32.
