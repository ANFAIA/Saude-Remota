Descripción detallada

Programa principal para medir frecuencia cardíaca, SpO2 y temperatura con un MAX30102.

Librería para medir la frecuencia cardíaca, saturación de oxígeno y temperatura, enviarlos a Firebase, mostrar en una pantalla OLED y permitir parada limpia del programa. Se utiliza un modelo de IA para calcular la precisión de los resultados y un nivel de riesgo.
